Hash#keys() → [Stringâ€¦]

Provides an Array containing the keys for items stored in the hash.

The order of the keys is not guaranteed.

Example
var h = $H({one: "uno", two: "due", three: "tre"});
h.keys();
// -> ["one", "three", "two"] (these may be in any order)
