class method Object.isString

View source on GitHub →

Object.isString(object) → Boolean
  • object (Object) – The object to test.

Returns true if object is of type String; false otherwise.

Examples
Object.isString("foo");
//-> true
 Object.isString("");
//-> true
 Object.isString(123);
//-> false
