String#unescapeHTML() → String

Strips tags and converts the entity forms of special HTML characters to their normal form.

Examples
'x &gt; 10'.unescapeHTML()
// -> 'x > 10'
 '<h1>Pride &amp; Prejudice</h1>;'.unescapeHTML()
// -> '<h1>Pride & Prejudice</h1>'
