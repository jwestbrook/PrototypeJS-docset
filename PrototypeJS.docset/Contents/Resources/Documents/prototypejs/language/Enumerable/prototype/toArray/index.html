Enumerable#toArray() → Array

Returns an Array containing the elements of the enumeration.

Example
$R(1, 5).toArray();
// -> [1, 2, 3, 4, 5]
 $H({ name: 'Sunny', age: 20 }).toArray();
// -> [['name', 'Sunny'], ['age', 20]]
